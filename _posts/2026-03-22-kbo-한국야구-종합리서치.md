## 1. 메이저 대회

### 1.1 KBO 한국시리즈

KBO 한국시리즈는 한국 프로야구 최고의 챔피언을 가리는 최종 결승 시리즈이다. 정규시즌과 포스트시즌을 거쳐 최종 2개 팀이 7전 4선승제로 맞붙는다.

| 연도 | 우승팀 | 상대팀 | 시리즈 전적 |
|------|--------|--------|------------|
| 2025 | LG 트윈스 | - | - |
| 2024 | KIA 타이거즈 | 삼성 라이온즈 | 4승 1패 |
| 2023 | LG 트윈스 | KT 위즈 | 4승 0패 |
| 2022 | SSG 랜더스 | 키움 히어로즈 | 4승 2패 |
| 2021 | KT 위즈 | 두산 베어스 | 4승 0패 |

| 순위 | 구단 | 우승 횟수 |

| 1 | KIA 타이거즈 (해태 포함) | 12회 |
| 2 | 삼성 라이온즈 | 8회 |

- 삼성 라이온즈는 한국시리즈 최다 진출(18회) 기록을 보유하고 있다.

## 2. 메이저 리그 (KBO 리그 구조)

### 2.1 리그 개요

- **정식 명칭**: 신한 SOL Bank KBO 리그 (타이틀 스폰서 포함)
- **창설**: 1982년 (6개 구단으로 출범)
- **참가 팀 수**: 10개 구단 (2015년 KT 위즈 가입 이후)
- **운영 기관**: KBO (한국야구위원회)

### 2.2 시즌 구조 (2026 기준)

| 구분 | 일정 | 비고 |
|------|------|------|
| 시범경기 | 3월 12일 ~ 3월 24일 | 팀당 12경기, 총 60경기 |
| 정규시즌 개막 | 3월 28일 | |
| 올스타전 | 7월 11일 | |
| 정규시즌 종료 | 9월 6일 | |
| 포스트시즌 | 9월 ~ 10월 | |

- **정규시즌**: 팀당 144경기, 총 720경기
- **시즌 기간**: 약 6개월

### 2.3 포스트시즌 구조

정규시즌 상위 5개 팀이 포스트시즌에 진출한다.

```
[정규시즌 4위] vs [정규시즌 5위]  -->  와일드카드 (단판 또는 2전제)
         |
    와일드카드 승자 vs [정규시즌 3위]  -->  준플레이오프 (5전 3선승제)
              |
         준PO 승자 vs [정규시즌 2위]  -->  플레이오프 (5전 3선승제)
              |
          PO 승자 vs [정규시즌 1위]  -->  한국시리즈 (7전 4선승제)
```

- 2025년부터 한국시리즈 홈 경기 편성 방식이 **2-3-2 방식**으로 변경되었다.
- 정규시즌 1위 팀은 한국시리즈 직행(어드밴티지 1승 없음, 홈 이점만 부여).

### 2.4 주요 규정

- 외국인 선수: 팀당 투수 2명 + 타자 1명 (총 3명)
- 2025년부터 피치클락(투구 시간 제한), 베이스 확대 등 새로운 규정 도입

## 3. 승격 시스템

### 3.1 KBO 퓨처스리그 (2군 리그)

KBO 퓨처스리그는 KBO 리그의 2군 리그로, 1군에 참가하는 10개 구단의 2군팀과 군(軍) 소속팀인 상무 피닉스 야구단이 참가하여 총 11개 팀으로 운영된다.

**리그 연혁**

| 시기 | 명칭 | 비고 |
|------|------|------|
| 1990년 ~ 2007년 | 2군 리그 | |
| 2008년 ~ 2014년 | 한국야구 퓨처스리그 | |
| 2015년 ~ 현재 | KBO 퓨처스리그 | 브랜드 아이덴티티 통합 |

**리그 구조**

- **남부 리그**: KIA, 삼성, 롯데, NC, 상무
- **북부 리그**: LG, 두산, SSG, KT, 키움, 한화
- 2025 시즌부터 **퓨처스리그 챔피언 결정전**을 신설하였다.
  - 북부리그 1위 vs 남부리그 2위, 북부리그 2위 vs 남부리그 1위가 4강전
  - 4강전 승자가 챔피언 결정전 진행

### 3.2 1군-2군 선수 이동 시스템

- KBO에는 유럽 축구식 팀 간 승격/강등(promotion/relegation) 시스템은 **존재하지 않는다**.
- 대신 **선수 단위**의 1군-2군 이동이 활발하게 이루어진다.
  - 1군 엔트리(등록 선수): 26명
  - 부상, 컨디션, 전략에 따라 1군-2군 간 수시 이동 가능
- **2차 드래프트**: 시즌 종료 후 다른 구단의 2군 선수를 지명하여 영입할 수 있는 제도

### 3.3 신인 드래프트

- 매년 시즌 종료 후 실시
- 고교, 대학, 독립리그, 해외 복귀 선수 등이 대상
- 1라운드부터 10라운드까지 진행 (전년도 하위팀부터 지명)

> 참고: [KBO 퓨처스리그 - 위키백과](https://ko.wikipedia.org/wiki/KBO_%ED%93%A8%EC%B2%98%EC%8A%A4%EB%A6%AC%EA%B7%B8), [KBO 퓨처스리그 - 나무위키](https://namu.wiki/w/KBO%20%ED%93%A8%EC%B2%98%EC%8A%A4%EB%A6%AC%EA%B7%B8), [KBO 퓨처스리그 공식](https://www.koreabaseball.com/futures/main.aspx)

## 4. 구단 비즈니스 모델 (BM)

### 4.1 수익 구조 개요

KBO 구단의 수익은 크게 다음 네 가지로 구성된다.

```
[구단 수익 구조]
  |
  +-- (1) 방송 중계권료
  +-- (2) 입장 수입 (티켓, F&B, MD)
  +-- (3) 모기업 지원금/광고비
  +-- (4) 스폰서십/마케팅 수입
```

### 4.2 방송 중계권

| 계약 기간 | 사업자 | 총 금액 | 연간 구단당 | 비고 |
|-----------|--------|---------|------------|------|
| 2024~2026 | 티빙(Tving) | 3년 1,350억원 | 약 45억원 | |
| 2027~2031 | 티빙(Tving) | 5년 4,500억원 | 약 90억원 (구단당 약 144억원 추정) | 종전 대비 2배 인상 |

- 20년 만에 중계권료가 약 45배 성장하였다.
- 2027년부터 구단당 연간 약 144억원의 고정 수입이 확보되었다.

### 4.3 입장 수입

| 연도 | 총 관중 수 | 관중 수입 |
|------|-----------|----------|
| 2024 | 1,088만 7,705명 | 1,500억원 이상 |

- 2024년 프로야구에 1,000만 관중 시대가 다시 열렸다.
- 티켓 수입 외에 구장 내 식음료(F&B), 굿즈(MD) 판매 수입도 포함된다.

### 4.4 모기업 지원 구조

| 구단 | 모기업(그룹) | 비고 |
|------|-------------|------|
| 삼성 라이온즈 | 삼성그룹 | |
| LG 트윈스 | LG그룹 | |
| 두산 베어스 | 두산그룹 | |
| 한화 이글스 | 한화그룹 | |
| 롯데 자이언츠 | 롯데그룹 | |
| KIA 타이거즈 | 현대자동차그룹(KIA) | |
| NC 다이노스 | 엔씨소프트 | |
| KT 위즈 | KT그룹 | |
| SSG 랜더스 | 신세계그룹 | |
| 키움 히어로즈 | 없음 (키움증권 네이밍 스폰서) | 유일한 독립 구단 |

- 키움 히어로즈를 제외한 9개 구단은 모기업(그룹)을 보유하고 있다.
- 2024년부터 모기업 지원 비율은 줄어들고, 자체 매출이 급상승하는 추세이다.
- KIA 타이거즈는 2024년에 85%라는 기록적인 자체 매출 비중을 달성하였다.

### 4.5 키움 히어로즈의 독립 경영 모델

- 키움증권과 네이밍 스폰서십 체결: 2024~2028년 5년간 최대 695억원 (연간 110~139억원)
- 모기업 없이 스폰서십 중심 경영 모델을 운영하는 유일한 구단

### 4.6 구단 가치 평가

- 포브스 구단가치평가에 따르면 KBO 구단들의 가치는 꾸준히 상승하고 있다.
- 자세한 구단별 가치평가는 포브스 보고서 참고.
