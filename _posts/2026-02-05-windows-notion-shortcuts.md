## 📌 단축키 확인 방법

Notion 내에서 `Ctrl + Shift + ?` 또는 `Ctrl + /` → "Shortcuts" 입력하면 전체 단축키 목록을 확인할 수 있습니다.

## 📌 팁

1. **템플릿 버튼**: `/template` 입력으로 템플릿 버튼 생성
2. **날짜 입력**: `@today`, `@tomorrow` 등으로 빠른 날짜 입력
3. **멘션**: `@` + 사용자 이름으로 멘션
4. **수식**: `/math` 또는 `$$` 로 수식 블록 생성

가장 자주 사용하는 단축키는 `Ctrl + P` (검색), `Ctrl + K` (링크), `/` (블록 생성) 입니다.
