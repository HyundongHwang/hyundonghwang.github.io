# 260312 Financial Modeling Prep API 리서치

작성 시각: 2026-03-12 01:34 KST  
리서치 대상: https://site.financialmodelingprep.com/

## 한 줄 결론

Financial Modeling Prep(FMP)는 미국 주식 중심의 시세, 재무제표, ETF, 뉴스, SEC, 거시지표를 한 벤더에서 폭넓게 제공하는 API 플랫폼이다.  
다만 당신이 원하는 범위를 기준으로 보면 `미국주식/미국ETF/미국채/SEC/뉴스`는 강하게 맞고, `한국주식`은 공식 사이트 인덱스 기준으로 지원 근거가 충분하지만, `한국ETF`와 특히 `한국채`는 공식 문서상 확인 근거가 약하거나 없어서 사전 실심볼 검증이 필요하다.

## 1. 소개

FMP는 주식, ETF, 재무제표, SEC, 13F, 뉴스, 경제지표를 API로 제공하는 금융 데이터 플랫폼이다.  
공식 홈페이지 기준으로 다음을 전면에 내세운다.

- 70,000+ 기업 커버리지
- 30년+ 데이터 이력
- 100+ API 엔드포인트
- 거래소 직접 소싱 데이터
- 실시간 및 과거 데이터
- SOC 2 Type II

즉, “하나의 API로 시세 + 펀더멘털 + 공시 + 뉴스 + 거시”를 묶어 쓰기 좋은 구조다.

## 2. 핵심 특징

### 2.1 데이터 범위가 넓다

공식 문서와 가격 페이지를 종합하면 FMP는 아래 계층을 동시에 제공한다.

- 가격 데이터: 실시간/근실시간 Quote, EOD, Intraday
- 기업 데이터: Profile, Screener, 검색, 상장심볼 목록
- 재무 데이터: 손익계산서, 재무상태표, 현금흐름표, Key Metrics, Ratios
- ETF/펀드 데이터: ETF 목록, ETF 보유종목, 국가 비중, 섹터 비중, 자산 익스포저
- 공시 데이터: SEC filings, filing extract, 13F 관련 데이터, transcripts
- 뉴스 데이터: Stock News, Press Releases, FMP Articles
- 거시 데이터: Treasury Rates, Economic Indicators, Economic Calendar

### 2.2 업데이트 주기가 명확하다

`developer/docs/cycle-times` 페이지에 업데이트 주기가 비교적 명확하게 공개되어 있다.

- Quote / Quote Short: 1초
- Historical Price EOD: 5분
- Financial Statements: 15분
- Market News: 1분
- Latest SEC Filings RSS Feed: 1분
- SEC Filings: 30분
- Insider Trades RSS Feed: 1분

실서비스에서 중요한 포인트는 “무엇이 얼마나 자주 갱신되는지”를 문서가 공개한다는 점이다.

### 2.3 요금제별 시장 범위 차이가 크다

가격 페이지상 핵심 차이는 단순 호출량보다 `시장 범위`다.

- `Basic`: 250 calls/day
- `Starter`: 주로 미국 커버리지
- `Premium`: 미국 + 영국/캐나다
- `Ultimate`: 글로벌 커버리지
- `Enterprise`: 라이선스/확장형

즉, 한국 시장까지 보려면 사실상 `Ultimate`를 전제로 봐야 한다.

## 3. 장점

- 시세, 재무, ETF, 뉴스, SEC, 거시를 한 API에서 묶어서 쓸 수 있다.
- 문서 구조가 비교적 일관적이고 Stable 엔드포인트 체계가 있다.
- 데이터 갱신 주기가 문서에 공개되어 있어 설계 판단이 쉽다.
- 미국 주식과 미국 ETF, SEC, 13F, earnings transcript 쪽은 특히 강하다.
- Screener, Symbol List, Search API가 있어 종목 검색과 파이프라인 구축이 쉽다.
- 글로벌 심볼도 공식 sitemap 인덱스에서 `.KS` 종목 엔트리가 다수 확인되어 한국 주식 지원의 간접 근거가 있다.
- Ultimate 플랜에서 WebSocket, bulk downloads, global coverage를 제공한다.

## 4. 단점

- 글로벌 지원이라고 해도 문서가 미국 자산 중심으로 구성되어 있다.
- 한국 ETF 지원은 공식 문서 예시가 희박해서 심볼 단위 검증이 필요하다.
- 한국 국채/회사채 데이터는 공식 문서에서 명확히 확인되지 않았다.
- “뉴스별 경제적 평가 점수” 같은 고수준 해석 레이어는 기본 제공이 약하다.
- 글로벌 범위를 쓰려면 사실상 Ultimate 이상이 필요해 비용이 빨리 올라간다.
- 데이터 표시/재배포는 일반 개인 요금제와 별개로 라이선스 검토가 필요하다.
- 일부 웹 페이지는 문서 탐색성이 좋지 않고, 실제 API는 키 없이는 바로 테스트하기 어렵다.

## 5. 주요 기능

### 5.1 가격/거래량

- Quote
- Quote Short
- Quote Change
- Historical Price EOD Full / Light
- Intraday 1m / 5m / 15m / 30m / 1h / 4h

이 조합이면 다음 요구는 충족된다.

- 현재가
- 시가/고가/저가/종가
- 거래량
- 일봉/분봉 히스토리

### 5.2 기업/종목 탐색

- Profile by Symbol
- Company Symbols List
- Search Symbol / Search Name
- Company Screener
- Exchange Variants

이건 미국주식뿐 아니라 글로벌 종목 탐색 파이프라인 구성에 유리하다.

### 5.3 재무 정보

- Income Statement
- Balance Sheet Statement
- Cash Flow Statement
- Key Metrics
- Metrics Ratios
- TTM 변형 및 bulk 변형

즉, 단순 시세 API가 아니라 펀더멘털 분석까지 가능한 구조다.

### 5.4 ETF / 펀드

- ETFs List
- Full ETF Quotes
- ETF Holder Bulk
- Country Weighting
- Sector Weighting
- ETF Asset Exposure
- Mutual Fund Disclosures

미국 ETF 분석에는 매우 적합하다.  
특히 holdings, country/sector weighting, asset exposure까지 붙어 있어 단순 가격 조회를 넘어 포트폴리오 분석용으로 쓸 수 있다.

### 5.5 뉴스 / 해석성 데이터

- Search Stock News
- Search Press Releases
- FMP Articles

이 조합은 “뉴스 수집”에는 충분히 강하다.  
다만 `각 뉴스에 대한 경제적 평가`를 구조화된 점수로 바로 주는지에 대해서는 공식 문서상 명확한 단일 엔드포인트를 확인하지 못했다.

대신 다음처럼 우회 설계는 가능하다.

- 일반 뉴스: `Search Stock News`
- 공식 발표: `Press Releases`
- FMP 자체 해설/분석성 콘텐츠: `FMP Articles`
- 별도 내부 로직: LLM/NLP로 impact score, sentiment score, macro relevance score 생성

즉, 뉴스 원천 데이터는 충분하지만 “해석 레이어”는 직접 얹는 쪽이 현실적이다.

### 5.6 SEC / 공시

- Latest Filings
- Search by Form Type
- Search by Symbol
- Search by CIK
- Filings Extract
- Filings Extract with Analytics by Holder
- Form 13F Filings Dates
- Search Transcripts
- Transcripts Dates by Symbol
- SEC Company Full Profile

이 부분은 FMP의 강점 중 하나다.  
미국 상장사 중심 분석, 공시 파싱, smart money 추적, earnings transcript 활용에는 꽤 적합하다.

### 5.7 거시 / 금리

- Treasury Rates
- Economics Indicators
- Economics Calendar

즉, 매크로 레이어를 별도 벤더 없이 붙일 수 있다.

## 6. 자세한 세부 기능들

### 6.1 가격 데이터 세부

실시간성 측면:

- Quote 류는 1초 단위 갱신
- 뉴스는 1분
- EOD는 5분

실무적으로는 다음 용도에 적합하다.

- 장중 대시보드
- 종목 모니터링
- 이벤트 감지
- 종가 기반 팩터/백테스트

### 6.2 펀더멘털 데이터 세부

펀더멘털 데이터는 최소한 아래 분석을 지원한다.

- 매출/영업이익/순이익 추세
- 자산/부채/자본 구조
- 영업/투자/재무 현금흐름
- 수익성, 밸류에이션, 효율성 비율
- TTM 기준 비교

따라서 “가격 + 거래량 + 각종 재무 정보들” 요구에는 미국주식 기준으로 매우 잘 맞는다.

### 6.3 ETF 데이터 세부

ETF 관련 데이터는 단순 가격 API보다 훨씬 깊다.

- ETF universe 조회
- ETF 실시간/근실시간 시세
- 보유 종목
- 국가별 비중
- 섹터별 비중
- 자산 노출 구조

이건 다음 같은 기능 구현에 직접 연결된다.

- ETF 룩스루 분석
- 특정 ETF의 국가/섹터 편향 분석
- ETF 간 스타일 비교
- 포트폴리오 중복도 분석

### 6.4 SEC 세부

단순 목록 조회가 아니라 extract 계열이 있다는 점이 중요하다.

- filing list 조회
- form type 기준 필터링
- symbol, cik 기준 검색
- filing 본문 추출
- holder analytics
- 13F 날짜 데이터
- transcript 검색

즉, “SEC 보고서 관련” 요구는 단순 충족이 아니라 꽤 강하게 충족하는 편이다.

### 6.5 거시 세부

거시 쪽은 적어도 다음 축을 제공한다.

- 국채 금리
- 경제지표 시계열
- 경제 캘린더

미국 자산과 뉴스/실적/금리 데이터를 한곳에서 합치기에 좋다.

## 7. 내가 필요한 기능 기준 적합도

### 7.1 미국주식

판정: `매우 적합`

근거:

- Quote, EOD, Intraday, 거래량 데이터 지원
- 재무제표, Key Metrics, Ratios 지원
- 뉴스, Press Releases, FMP Articles 지원
- SEC filings, transcripts 지원
- 업데이트 주기 공개

### 7.2 한국주식

판정: `유력하지만 최종 확정 전 PoC 필요`

확인 근거:

- 가격 페이지가 Ultimate에 `Global Coverage`를 명시
- 공식 sitemap에 `.KS` 심볼 엔트리가 다수 존재
- 예시로 `005930.KS`, `005380.KS`, `000660.KS`가 sitemap 인덱스에서 확인됨

해석:

공식 문서가 한국 시장을 전면 사례로 설명하지는 않지만, `Global Coverage` 명시와 공식 sitemap의 `.KS` 엔트리를 함께 보면 한국 주식 지원 가능성은 높다고 해석하는 것이 타당하다. 다만 개별 웹 페이지 접근은 일관되지 않아 실제 API 응답 검증이 꼭 필요하다.

주의:

- 실제 사용 전 반드시 대표 심볼 20~30개로 quote, historical, financial statements, volume을 샘플 검증해야 한다.

### 7.3 미국채

판정: `부분 적합`

확인 근거:

- `Treasury Rates` API 존재
- `Search CUSIP`, `Company Notes`, bond 관련 검색 경로 존재

해석:

- 미국 국채 금리 시계열은 대응 가능
- 하지만 블룸버그식 개별 채권 가격/수익률/듀레이션/쿠폰/만기구조를 광범위하게 다루는 “본격 채권 데이터 벤더” 수준으로 문서화되어 있지는 않다

따라서:

- `미국채 금리/커브 관찰` 목적이면 적합
- `개별 미국채 종목 데이터 터미널 수준` 목적이면 부족할 수 있다

### 7.4 한국채

판정: `공식 근거 부족, 사실상 미확인`

리서치 결과:

- 공식 문서에서 한국 국채/한국 회사채 지원을 명시한 페이지를 찾지 못했다
- South Korea bond universe 관련 공식 예시를 찾지 못했다

보수적 결론:

한국채가 핵심 요구사항이면 FMP를 단독 메인 벤더로 선택하기 전에 반드시 PoC가 필요하다.  
현재 확보한 공식 근거만으로는 “지원한다”고 말하기 어렵다.

### 7.5 미국ETF

판정: `매우 적합`

근거:

- ETF list
- ETF quotes
- holdings
- country weighting
- sector weighting
- asset exposure

단순 ETF 가격 조회를 넘어 ETF 분석 플랫폼을 만들기 좋은 데이터 구성을 갖고 있다.

### 7.6 한국ETF

판정: `가능성은 있으나 공식 근거 약함`

확인된 사실:

- Ultimate의 글로벌 커버리지는 확인됨
- ETF 관련 API 자체는 풍부함

확인되지 않은 부분:

- 한국 ETF 심볼 예시나 공식 문서 사례를 이번 조사에서 직접 찾지 못함
- 한국 ETF 페이지 인덱스는 공식 sitemap에서 명확히 확인하지 못함

보수적 결론:

한국 ETF가 중요하면 실제 심볼 예시를 넣어 다음 4가지를 먼저 테스트해야 한다.

- Quote 존재 여부
- Historical price 존재 여부
- ETF quote 존재 여부
- holdings / weighting 계열 지원 여부

### 7.7 가격, 거래량, 각종 재무 정보들

판정: `매우 적합`

미국주식 기준:

- 가격: 충분
- 거래량: 충분
- 분봉/일봉: 충분
- 재무제표: 충분
- 지표/비율: 충분

한국주식 기준:

- 지원 근거는 높지만 범위/품질은 샘플 검증 권장

### 7.8 경제뉴스, 각 뉴스에 대한 경제적 평가

판정: `뉴스 수집은 적합, 평가 자동화는 부분 적합`

확인된 사실:

- Stock News
- Press Releases
- FMP Articles
- Economics Calendar
- Economics Indicators

해석:

FMP는 뉴스 원문/요약/분류/경제 이벤트 인접 데이터는 제공하는 편이다.  
하지만 “각 뉴스별 경제적 영향도 점수”나 “정규화된 경제 해석 레이어”는 공식 문서상 명확히 제품화되어 있지 않다.

실무적 권장:

- FMP를 뉴스 ingestion source로 사용
- 내부에서 별도 평가 레이어를 추가
- LLM 또는 룰 기반으로 다음 점수를 생성
  - macro impact
  - earnings impact
  - sector impact
  - risk sentiment

### 7.9 SEC 보고서 관련

판정: `매우 적합`

이건 FMP가 가장 잘 맞는 영역 중 하나다.

- latest filings
- by form type
- by symbol
- by CIK
- filing extract
- 13F 관련 데이터
- transcript 검색

미국 상장사 리서치 자동화에는 상당히 유용하다.

## 8. 어떤 요금제가 필요한가

### 8.1 당신 요구 기준 최소 추정

`미국주식 + 미국ETF + 미국채 + 뉴스 + SEC`만 쓰면 Starter 또는 Premium의 일부로 시작 가능성이 있다.  
하지만 `한국주식 + 한국ETF`를 넣는 순간 현실적으로 `Ultimate`를 보는 것이 맞다.

### 8.2 추천 판단

- 한국 시장이 중요하지 않다: Starter/Premium 검토 가능
- 한국 시장이 중요하다: Ultimate 우선 검토
- 외부 고객에게 데이터 표시/재배포 예정이다: Enterprise 또는 별도 라이선스 문의 필요

## 9. 최종 판단

### 9.1 FMP가 잘 맞는 경우

- 미국 주식 리서치 자동화
- 실적/공시/SEC 중심 워크플로우
- ETF holdings 기반 분석
- 뉴스 + 거시 + 기업 펀더멘털 결합
- 빠르게 MVP를 만들고 싶은 경우

### 9.2 FMP가 애매한 경우

- 한국채가 핵심인 경우
- 한국ETF 커버리지가 반드시 확실해야 하는 경우
- 뉴스별 경제영향 점수를 바로 제품 기능으로 써야 하는 경우
- 채권 전문 데이터가 필요한 경우

### 9.3 내 결론

당신의 요구사항 전체를 기준으로 보면 FMP는 `미국 자산 + SEC + 뉴스 + 기본 거시`에는 매우 강하고, `한국 주식`도 유력하게 지원한다.  
하지만 `한국채`와 `한국ETF`는 공식 문서 근거가 약하므로, 메인 벤더로 확정하기 전 다음 PoC를 반드시 권장한다.

1. 한국 주식 30개 심볼 quote/historical/statements 검증
2. 한국 ETF 10개 심볼 quote/ETF 데이터 검증
3. 미국채에서 필요한 수준이 금리 시계열인지, 개별 채권 종목 데이터인지 정의
4. 뉴스별 경제 평가를 내부 생성할지, 외부 별도 벤더를 붙일지 결정

보수적으로 말하면:

- 미국 중심 리서치 API로는 좋다
- 한국 주식까지는 꽤 가능성이 높다
- 한국 ETF/한국채까지 완전 커버를 기대하면 아직 불확실성이 있다

## 10. 사실 검증 메모

이번 리서치는 공식 홈페이지, 공식 가격 페이지, 공식 개발문서, 공식 cycle times, 공식 sitemap을 교차 검증했다.

확인된 사실:

- FMP는 실시간/과거 시세, 재무제표, ETF, 뉴스, SEC, 거시 API를 제공한다
- Ultimate 플랜은 글로벌 커버리지를 명시한다
- `.KS` 심볼 엔트리가 공식 sitemap에 실제로 존재한다
- Treasury Rates, Economics Indicators, Economics Calendar, SEC filings 관련 문서가 존재한다

끝까지 공식 문서에서 확인하지 못한 것:

- 한국 국채/회사채 지원 명시
- 한국 ETF에 대한 명시적 사례
- 각 뉴스별 정규화된 경제 영향 점수 API

또한 한국 종목 관련 개별 웹 페이지는 라이브 접근 시 일관되게 열리지 않았고, 공식 sitemap 인덱스에서만 강한 흔적이 확인되었다. 따라서 한국 주식은 “가능성 높음”, 한국 ETF/한국채는 “공식 근거 부족”으로 보수적으로 판단했다.
