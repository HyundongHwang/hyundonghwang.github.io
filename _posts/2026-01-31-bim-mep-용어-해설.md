## 📌 1. ConnectionState - 연결 상태

BIM MEP 모델에서 요소 간의 연결 관계를 나타내는 상태값입니다. 시스템의 무결성을 확인하고 충돌 검토(Clash Detection)에 활용됩니다.

### 🔹 상태 값

| 값 | 설명 | 실무 적용 |
|---|------|----------|
| **Connected** | 두 아이템이 올바르게 연결됨 | 배관, 덕트, 케이블트레이 등의 요소가 설계 의도대로 정상 연결된 상태 |
| **Clash** | 두 아이템이 충돌/간섭함 | 서로 다른 시스템이 같은 공간을 점유하여 시공 불가능한 상태 |
| **None** | 연결 관계 없음 | 독립적인 요소이거나 아직 연결되지 않은 상태 |

### 🧠 충돌 검토(Clash Detection)의 중요성

```
┌─────────────────────────────────────────────────────────┐
│                    BIM 충돌 검토 워크플로우                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   [설계 모델] ──→ [통합 모델] ──→ [충돌 검토]              │
│       │              │              │                   │
│       ▼              ▼              ▼                   │
│   건축/구조      MEP 통합      Clash Report             │
│   MEP 각 분야                  - Hard Clash             │
│                                - Soft Clash             │
│                                - Clearance              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

- **Hard Clash**: 물리적으로 같은 공간을 점유하는 충돌
- **Soft Clash**: 설치/유지보수 공간 부족
- **Clearance Clash**: 최소 이격거리 미확보

## 🏢 2. MepType - MEP 요소 유형

MEP 시스템을 구성하는 요소들의 분류 체계입니다. 각 타입은 고유한 특성과 연결 규칙을 가집니다.

### 🔹 유형 분류

| 값 | 이름 | 설명 | 예시 |
|---|------|------|------|
| 0 | **Line** | 선형 요소 | 파이프, 덕트, 케이블트레이 |
| 1 | **Fitting** | 피팅 | 엘보우, 티, 리듀서 |
| 2 | **Accessory** | 액세서리 | 밸브, 댐퍼, 스트레이너 |
| 3 | **AirTerminal** | 송풍구 | 디퓨저, 그릴, 레지스터 |
| 4 | **Lighting** | 조명 | 형광등, LED 조명, 비상등 |
| 5 | **PlumbingFixtures** | 위생기구 | 세면대, 양변기, 소변기 |
| 7 | **ElectricalFixtures** | 전기기구 | 콘센트, 스위치, 분전반 |
| 8 | **Equipment** | 장비 | 펌프, 팬, 칠러, AHU |
| 9 | **Sprinkler** | 스프링클러 | 폐쇄형, 개방형 헤드 |
| 99 | **Etc** | 기타 | 분류되지 않은 기타 요소 |

### 🏢 MEP 시스템 구성도

```
┌──────────────────────────────────────────────────────────────────┐
│                      MEP 시스템 요소 체계                          │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐       │
│  │Equipment│───▶│  Line   │───▶│ Fitting │───▶│Terminal │       │
│  │ (장비)   │    │ (배관)   │    │ (피팅)  │    │ (말단)   │       │
│  └─────────┘    └─────────┘    └─────────┘    └─────────┘       │
│       │              │              │              │             │
│       ▼              ▼              ▼              ▼             │
│  - 칠러         - 파이프        - 엘보우       - 디퓨저          │
│  - 펌프         - 덕트          - 티           - 그릴            │
│  - AHU          - 케이블트레이   - 리듀서       - 스프링클러       │
│  - 보일러                                      - 조명            │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### 🔹 주요 장비(Equipment) 설명

**칠러(Chiller)**: 냉수를 생산하여 공조 시스템에 공급하는 핵심 장비. 공랭식과 수냉식으로 구분됩니다.

**AHU(Air Handling Unit)**: 공기조화기. 냉각/가열 코일과 팬을 포함하여 공기를 조절하고 덕트를 통해 각 공간에 분배합니다. 일반적으로 2,000~100,000 CFM 용량으로 대규모 공간에 적용됩니다.

**FCU(Fan Coil Unit)**: 팬코일유닛. 개별 공간의 냉난방을 담당하는 소형 장치로, AHU와 달리 외기를 처리하지 않습니다.

## 📌 3. FittingType - 피팅 유형

배관, 덕트, 케이블트레이에서 방향 전환, 분기, 크기 변경 등에 사용되는 연결 부품입니다.

### 🔹 피팅 유형 상세

| 유형 | 한글명 | 설명 | 적용 분야 |
|------|--------|------|----------|
| **Reducer** | 리듀서 | 관경 축소 (동심/편심) | 배관, 덕트 |
| **Elbow** | 엘보우 | 방향 변경 (45°/90°) | 배관, 덕트 |
| **VerticalTeeUp** | 수직 티 (위) | 수직 상향 분기 | 배관 |
| **Cross** | 크로스 | 십자형 4방향 연결 | 배관, 덕트 |
| **Tee** | 티 | T자형 3방향 분기 | 배관, 덕트 |
| **Wye** | 와이 | Y자형 45° 분기 | 덕트, 배관 |
| **Coupling** | 커플링 | 동일 규격 직선 연결 | 배관 |
| **Coupling_GrooveJoint** | 그루브 조인트 | 홈 접합 방식 커플링 | 소방배관 |
| **EndCap** | 엔드캡 | 관 끝단 마감 | 배관 |
| **Cap** | 캡 | 캡 마감 | 배관 |
| **Tap** | 탭 | 분기 연결부 | 덕트 |
| **HorizontalBend** | 수평 벤드 | 수평면 방향 전환 | 케이블트레이 |
| **VerticalInsideBend** | 수직 내부 벤드 | 내측 수직 굴곡 | 케이블트레이 |
| **VerticalOutsideBend** | 수직 외부 벤드 | 외측 수직 굴곡 | 케이블트레이 |
| **BlindFlange** | 블라인드 플랜지 | 관 끝단 밀폐 | 배관 |
| **Flange** | 플랜지 | 볼트 체결 연결 | 배관 |

### 🔹 엘보우(Elbow) 상세

엘보우는 유체 흐름의 방향을 변경하는 핵심 피팅입니다.

```
          Long Radius (LR)              Short Radius (SR)
          R = 1.5 × D                   R = 1.0 × D

              ╭────╮                        ╭──╮
              │    │                        │  │
              │    │                        │  │
         ─────╯    ╰─────               ────╯  ╰────

    • 낮은 압력손실                    • 협소한 공간에 적합
    • 부드러운 흐름                    • 높은 압력손실
    • 일반 배관에 권장                 • 유지보수 공간 제한시 사용
```

### 🔹 리듀서(Reducer) 유형

```
    Concentric Reducer              Eccentric Reducer
    (동심 리듀서)                    (편심 리듀서)

         ╲      ╱                        ╲      │
          ╲    ╱                          ╲     │
           ╲  ╱                            ╲    │
            ╲╱                              ╲   │
            │                                ╲  │
            │                                 ╲ │

    • 중심축 유지                     • 한쪽 면 평면 유지
    • 수직 배관에 사용                • 수평 배관에 사용 (배수 고려)
    • 가스/증기 배관                  • 펌프 흡입측
```

### ⚖️ 티(Tee)와 와이(Wye)의 차이

- **Tee**: 90° 분기로 메인 라인에서 직각으로 분기. 균등한 유량 분배가 필요할 때 사용
- **Wye**: 45° 분기로 압력 손실이 적음. 메인 덕트 크기가 변경될 때 권장

## 📌 4. AccessoryType - 액세서리 유형

배관 및 덕트 시스템에서 제어, 측정, 보호 기능을 수행하는 부속품입니다.

### 🔹 액세서리 유형 상세

| 유형 | 한글명 | 기능 | 설치 위치 |
|------|--------|------|----------|
| **Valve** | 밸브 | 유량 제어/차단 | 장비 전후, 분기점 |
| **Strainer** | 스트레이너 | 이물질 여과 | 밸브/장비 상류측 |
| **SteamTrap** | 스팀 트랩 | 응축수 배출, 증기 차단 | 증기배관 하부 |
| **FlowMeter** | 유량계 | 유량 측정 | 직관부 확보 구간 |
| **Gauge** | 게이지 | 압력/온도 측정 | 장비 출입구 |
| **AirVent** | 공기 벤트 | 배관 내 공기 배출 | 배관 최상부 |
| **Damper** | 댐퍼 | 풍량 제어 | 덕트 분기부 |
| **Chamber** | 챔버 | 압력 완충/소음 저감 | 팬 출구측 |

### 🔹 밸브(Valve) 종류

```
┌─────────────────────────────────────────────────────────────────┐
│                        밸브 종류 및 용도                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  게이트 밸브 (Gate)     : 완전 개폐용, 직선 유로                   │
│  글로브 밸브 (Globe)    : 유량 조절용, 압력손실 큼                  │
│  볼 밸브 (Ball)         : 신속 개폐, 직선 유로                     │
│  버터플라이 밸브        : 대구경, 공간 절약                        │
│  체크 밸브 (Check)      : 역류 방지                               │
│  안전 밸브 (Safety)     : 과압 방지, 자동 개방                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 🔹 스트레이너(Strainer)

배관 내 스케일, 녹, 용접 잔재물 등을 걸러내어 밸브, 펌프, 트랩 등 주요 기기를 보호합니다.

- **Y-Strainer**: 가장 일반적인 형태, 연속 운전 시스템에 적합
- **Basket Strainer**: 대용량 이물질 처리, 쉬운 청소

**설치 원칙**: 스트레이너는 모든 스팀트랩, 유량계, 제어밸브의 상류측에 설치해야 합니다.

### 🔹 댐퍼(Damper) 종류

| 유형 | 용도 | 작동 방식 |
|------|------|----------|
| **VCD (Volume Control Damper)** | 풍량 조절 | 수동/자동(MVCD) |
| **Fire Damper** | 화재 시 연기/불꽃 차단 | 퓨즈블 링크 (열감지) |
| **Smoke Damper** | 연기 감지 차단 | 연기 감지기 연동 |
| **Backdraft Damper** | 역류 방지 | 자동(압력차) |
| **Balancing Damper** | 시스템 밸런싱 | 수동 조정 |

## 📌 5. ModelWork - 모델 작업 분류

BIM 프로젝트에서 각 분야별 작업 영역을 구분하는 분류 체계입니다.

### 🔹 분류 체계

| 값 | 한글명 | 설명 | 포함 요소 |

| **Architecture** | 건축 | 건축 마감 | 벽체, 바닥, 천장, 창호 |

| **CableTrays** | 케이블트레이 | 케이블 지지대 | 래더, 채널, 솔리드 타입 |
| **RaceWay** | 레이스웨이 | 전선관로 | 케이블 보호관 |
| **Pipes** | 배관 | 유체 이송관 | 급수, 급탕, 냉온수, 소화 |

## 📌 6. SectionType - 단면 유형

덕트 및 케이블트레이의 단면 형상을 정의합니다. 설치 공간, 효율성, 비용에 따라 선택됩니다.

### 🔹 단면 유형

| 유형 | 한글명 | 특징 | 주요 적용 |
|------|--------|------|----------|
| **Rectangular** | 직사각형 | 공간 활용도 높음 | 상업/오피스 천장 공간 |
| **Circular** | 원형 | 최적의 공기역학 | 산업용, 스파이럴 덕트 |
| **Oval** | 타원형 | 원형과 직사각형의 절충 | 제한된 천장 공간 |
| **Ladder** | 래더 | 케이블트레이 개방형 | 일반 케이블 지지 |
| **Channel** | 채널 | 케이블트레이 U자형 | 소규모 케이블 런 |

### ⚖️ 덕트 단면 형상 비교

```
    Rectangular            Circular              Oval
    (직사각형)              (원형)                (타원형)

    ┌──────────┐            ╭──╮               ╭────────╮
    │          │           ╱    ╲             ╱          ╲
    │          │          │      │           │            │
    │          │          │      │           │            │
    │          │           ╲    ╱             ╲          ╱
    └──────────┘            ╰──╯               ╰────────╯

    장점:                  장점:               장점:
    • 공간 효율적          • 최저 압력손실      • 낮은 높이
    • 제작 용이            • 정숙 운전         • 원형보다 공간 효율
    • 설치 편의성          • 누기 최소         • 직사각보다 효율적

    단점:                  단점:               단점:
    • 높은 압력손실        • 높이 필요         • 제작비 높음
    • 소음 발생            • 천장 공간 제약    • 피팅 제작 어려움
    • 누기 가능성          • 커스텀 피팅       • 공급업체 제한
```

### 🔹 케이블트레이 단면 유형

```
    Ladder Type             Channel Type          Solid Bottom
    (래더형)                (채널형)              (솔리드형)

    ┌─┐     ┌─┐            ┌───────────┐         ┌───────────┐
    │ │ ─── │ │            │           │         │███████████│
    │ │     │ │            │           │         │███████████│
    │ │ ─── │ │            │           │         │███████████│
    │ │     │ │            └───────────┘         └───────────┘
    └─┘ ─── └─┘

    • 75% 시장 점유         • 소규모 케이블런     • EMI 차폐
    • 최적 방열             • 협소 공간          • 분진/습기 차단
    • 습기 배출             • 분기 케이블        • 민감 회로 보호
```

**래더(Ladder) 케이블트레이**: 가장 보편적인 형태로, 사다리 모양의 구조입니다. 우수한 통풍과 방열 특성을 가지며, 전체 케이블트레이 설치의 약 75%를 차지합니다. 렁(Rung) 간격은 6", 9", 12", 18" 옵션이 있습니다.

**채널(Channel) 케이블트레이**: U자형 단면의 소형 트레이로, 분기 케이블이나 소규모 케이블 런에 적합합니다. 컨딧 사용이 불편한 경우의 대안으로 활용됩니다.

## 📌 7. LineType - 라인 유형

배관 및 덕트 라인의 물리적 특성을 정의합니다.

### 🔹 라인 유형

| 유형 | 한글명 | 특성 | 적용 분야 |
|------|--------|------|----------|
| **Regular** | 일반 라인 | 강성(Rigid), 직선형 | 주 배관/덕트 |
| **Flexible** | 유연 라인 | 플렉시블, 굴곡 가능 | 장비 연결, 진동 흡수 |

### ⚖️ Rigid vs Flexible 비교

```
┌─────────────────────────────────────────────────────────────────┐
│                  Rigid(강성) vs Flexible(유연) 비교              │
├─────────────────────────┬───────────────────────────────────────┤
│       Rigid Duct        │           Flexible Duct               │
├─────────────────────────┼───────────────────────────────────────┤
│ • 갈바나이즈드 스틸      │ • 와이어 코일 + 플라스틱 + 단열재      │
│ • 높은 내구성            │ • 설치 용이, 저렴                      │
│ • 부드러운 내면          │ • 굴곡 가능                            │
│ • 효율적 공기 흐름       │ • 진동 흡수, 소음 저감                 │
│ • 청소 용이              │ • 굴곡부 압력손실                      │
│ • 장기 사용에 적합       │ • 처짐 발생 가능                       │
├─────────────────────────┴───────────────────────────────────────┤
│                        권장 사용 패턴                            │
│                                                                 │
│   [AHU] ══════ Rigid Trunk Line ══════╤══════════════╤═══       │
│                                       │              │          │
│                                  Flex duct      Flex duct       │
│                                  (≤10ft)       (≤10ft)          │
│                                       │              │          │
│                                   [Diffuser]    [Diffuser]      │
│                                                                 │
│   ※ 메인 트렁크는 Rigid, 말단 분기는 Flexible (10ft 이내)         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### ⚠️ 플렉시블 덕트 사용 시 주의사항

1. **최소 굴곡 반경 유지**: 급격한 굴곡은 압력손실 증가
2. **길이 제한**: 일반적으로 10ft(약 3m) 이내 권장
3. **처짐 방지**: 적절한 지지대 설치 필요
4. **단열 확인**: 결로 방지를 위한 단열 상태 점검
