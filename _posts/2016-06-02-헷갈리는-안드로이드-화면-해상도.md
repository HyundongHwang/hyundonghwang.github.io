# 안드로이드의 3가지 해상도 범주
- http://stardust328.tistory.com/entry/%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C-%ED%95%B4%EC%83%81%EB%8F%84-%EA%B7%9C%EA%B2%A9

- LDPI
    - Density(인치당 픽셀수) : 120 , dp = px * 1.33
- MDPI
    - Density : 160, dp = px * 1.0
- HDPI
    - Density : 240, dp = px * 0.66
