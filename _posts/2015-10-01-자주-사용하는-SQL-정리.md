## 1. 서론

주로 클라개발을 해서 SQL문을 항상 대강 알고 사용하다가 요즘 시간이 좀 나서 자주쓰는 구문 중심으로 제가 참고할 목적으로 정리해 봤습니다.

### 3-8. 그룹핑하고 카운트를 구함
> GROUP BY
> COUNT 함수 사용

```
SELECT vend_id, COUNT(*) as num_prod
FROM Products
GROUP BY vend_id;
```

```text
vend_id    num_prod
---------- -----------
BRS01      3
DLL01      4
FNG01      2
```

### 3-9. 그룹핑하고 카운트를 구하며, 그 그룹의 조건으로 필터링
> GROUP BY
> COUNT 함수
> HAVING
> * HAVING은 그룹에 대한 WHERE라고 생각하면 된다.


```sql
SELECT vend_id, COUNT(*) as num_prod
FROM Products
WHERE prod_price >= 4
GROUP BY vend_id
HAVING COUNT (*) >= 2;
```

```text
vend_id    num_prod
---------- -----------
BRS01      3
FNG01      2
```

### 3-13. 암시적인 INNER JOIN 에 컬럼이름 커스토마이즈 

```sql
SELECT c. cust_name AS '고객이름', c .cust_contact AS '고객주소'
FROM Customers AS c, Orders AS o, OrderItems AS oi
WHERE c. cust_id = o .cust_id
    AND oi .order_num = o.order_num
    AND prod_id = 'RGAN01' ;
```

AS
    DECLARE @order_num INTEGER;

    FROM Orders ;

    INSERT INTO Orders( order_num, order_date , cust_id)
